# ☕ Кафе «Арома» — сайт-визитка

Лендинг (одностраничный сайт) для кафе: главный экран, меню, блок «О нас»,
форма брони столика и подвал с контактами.

## Технологии
- **HTML** — структура страницы
- **CSS** — оформление, адаптивность под телефоны
- **JavaScript** — обработка формы брони

## Как запустить
Открыть файл `index.html` в любом браузере (двойной клик).
Или, для удобной разработки, через расширение VS Code «Live Server».

## Структура
| Файл | Назначение |
|------|------------|
| `index.html` | скелет страницы (что где находится) |
| `styles.css` | внешний вид (цвета, шрифты, отступы) |
| `script.js`  | поведение (обработка формы) |
